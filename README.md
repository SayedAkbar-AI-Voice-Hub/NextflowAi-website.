# NextFlow AI - Local Business Websites

Professional websites for local service businesses including HVAC, plumbers, landscapers, electricians, and more. This platform helps small to medium-sized businesses establish their online presence with lead generation capabilities and appointment booking.

## Features

- **Professional Website Design** - Modern, mobile-responsive design that builds trust
- **Lead Capture Forms** - Smart contact forms that capture customer information
- **Online Booking System** - Let customers book appointments 24/7
- **Multi-Industry Support** - Built for HVAC, plumbing, landscaping, electrical, roofing, cleaning services, and more

## Tech Stack

- **React 19** - Latest React with improved performance
- **TypeScript** - Full type safety
- **Vite** - Lightning-fast build tool and dev server
- **TailwindCSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icons
- **Cal.com** - Embedded booking calendar

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd "nextflowai- website design"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3001](http://localhost:3001) in your browser.
